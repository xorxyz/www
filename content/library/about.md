`xor` is an open world, multiplayer online computer security puzzle  game.

Players take control of a virtual machine, which they can use to explore virtual networks and interact with an underlying real-world simulation. Solving puzzles reward players with currency and points, which they can use to upgrade the machine and access different pieces of software.

Every unlocked piece of emulated hardware and software makes 
itself available in the level editor, which allows players 
to create puzzles for each other. 

Solving challenges gives you experience points.
Experience points give you levels.
Levels give you ability and skill points.
Abilities points can be used to ugrade abilities.
Skill points can be used to ugrade skills.

At the moment, `xor` is made up of a few moving parts:

1. a system for automating deployment, scaling, and management of emulated hardware
2. a distributed event bus
3. a simulated environment you can run tests against
4. a framework to build bots that can evolve inside a simulated environment
