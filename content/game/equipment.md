Characters can equip weapons, vehicles and accessories.

```

weapons:
  - editors         # staff
  - packet tools    # riffle
  - infra tools     # wrench
  - exploit tools   # cards

vehicles:
  - runtimes        # boots
  - logs            # truck
  - routing         # motorcycle
  - infrastructure  # van

accessories:
  - shells          # shell
  - jobs            # toolbox
  - daemons         # lockpicks
  - storage         # drive

```
